Política de Privacidade do apiyoutube2021

Última atualização: 2025-02-26

Obrigado por usar o apiyoutube2021, um addon desenvolvido para o Kodi que permite acessar conteúdo do YouTube. Esta política de privacidade descreve como coletamos, usamos e protegemos suas informações ao utilizar este addon.

1. Informações Coletadas

Este addon utiliza a API do YouTube para permitir o acesso a vídeos e playlists. Para acessar a API do YouTube, o addon solicita permissão para usar sua conta do Google. Isso inclui acesso limitado às suas informações do YouTube (como playlists e histórico de visualizações). Nenhuma informação pessoal é armazenada ou compartilhada pelo addon.

2. Uso das Informações

As informações coletadas pela API do YouTube são usadas exclusivamente para fornecer as funcionalidades do addon. Não utilizamos suas informações para fins comerciais ou de marketing.

3. Compartilhamento de Informações

Suas informações não são compartilhadas com terceiros, exceto quando exigido pela API do YouTube.

4. Segurança

Este addon segue as práticas recomendadas de segurança para proteger seus dados durante o uso da API do YouTube. No entanto, lembre-se de que nenhum método de transmissão pela internet ou armazenamento eletrônico é 100% seguro.

5. Alterações nesta Política

Podemos atualizar esta política de privacidade periodicamente. Recomendamos que você revise esta página regularmente para se manter informado sobre quaisquer alterações.
